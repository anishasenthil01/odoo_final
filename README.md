# Cafe POS System

A complete Point of Sale system for cafes and restaurants.

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** FastAPI + Python
- **Database:** PostgreSQL / SQLite

## Features
- POS System with cart and payment
- Kitchen Display System (KDS)
- Customer Self-Order Kiosk
- Admin Dashboard with full management
- Staff Dashboard
- Customer Display
- UPI QR Code payments
- Coupon & discount system
- Table management
- Order history & reviews
